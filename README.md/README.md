# 🍊 Vita-c AI: Autonomous Research Agent

Vita-c is a professional AI research tool that uses **Llama 3** (via Ollama) and **Tavily Search** to perform deep-dive research on any topic in real-time.

### 🚀 Features
- **Local LLM:** Powered by Llama 3 for privacy and speed.
- **Web Intelligence:** Real-time data fetching using Tavily API.
- **Executive Reports:** Automatically synthesizes complex data into concise reports.

### 🛠️ Tech Stack
- Python
- Streamlit (UI)
- LangChain
- Ollama
